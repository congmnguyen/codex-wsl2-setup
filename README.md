# codex-wsl-setup

This repository contains my current Codex CLI configuration for a WSL-based setup.

Files:
- `config.toml`: active Codex configuration exported from `~/.codex/config.toml`
- `wsl-image-paste.md`: notes for fixing `Alt+V` image paste on WSL
- `codex-notify.md`: Windows notification setup for "turn complete" on WSL2

Notes:
- Project trust rules and MCP server settings are machine-specific.
- Update the local paths before using this config on another machine.
